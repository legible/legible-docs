A JSON based [ONIX](https://en.wikipedia.org/wiki/ONIX_for_Books) metadata schema, converted from XML using [OnixEdit](https://onixedit.com/en-us/Support/Resources). We will be handling all business logic using the JSON metadata format, and this will be stored in a MongoDB database.

<Note>
  If you are looking for information about the database go to our [Content
  Metadata](/architecture/databases/content-metadata) page.
</Note>
